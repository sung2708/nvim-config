# My config nvim i using in company

This repository contains my Neovim configuration that I use in my company for development. It includes various plugins and settings tailored to enhance productivity and streamline the coding workflow.

## Features

- Syntax highlighting for multiple programming languages
- Autocompletion and code snippets
- File explorer integration

## Installation

To use this configuration, follow these steps:

1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/sung2708/nvim-config_win.git
   ```
2. I using vimplug as plugin manager, so make sure you have it installed. You can find the installation instructions [here](https://github.com/junegunn/vim-plug).
3. Copy the contents of the cloned repository to your Neovim configuration directory:
   - For Linux/Mac: `~/.config/nvim/`
   - For Windows: `%USERPROFILE%\AppData\Local\nvim\`
4. Open Neovim and run the following command to install the plugins:
   ```vim
    :PlugInstall
   ```
5. Restart Neovim to apply the changes.

## Usage

Once the configuration is set up, you can start using Neovim with the enhanced features provided
by the plugins and settings included in this configuration.
Feel free to customize the configuration files to suit your preferences and workflow.
